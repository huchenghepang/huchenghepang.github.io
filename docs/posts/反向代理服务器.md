## 3. 启动`nginx`服务器：

```
service nginx start
// 出现：Redirecting to /bin/systemctl start nginx.service 表示启动成功
```

## 4. 关闭`nginx`服务器

   ①：通过系统服务管理器（Systemd）:

   ```
   systemctl stop nginx
   ```

   ②：如果Nginx没有设置为系统服务，可以直接使用`nginx`命令:

   ```
   nginx -s stop
   ```

   ③：如果上述方法都不适用，可以发送信号直接杀死nginx进程（慎用，可能导致数据丢失）:

   ```
   kill -quit xxxx
   xxxx是进程的编码
   pkill -9 nginx
   强制杀死 会导致数据丢失
   
   
   ```
