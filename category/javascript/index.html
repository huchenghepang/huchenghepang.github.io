时间: 5/17/2024分类: JavaScript标签: 前端

1. JSDoc 简介

JSDoc 是一种用于 JavaScript 注释的标准，可以帮助开发者描述代码的功能、参数、返回值等信息，并提供代码补全、参数提示和文档生成等功能。

2. JSDoc 注释格式

JSDoc 注释以 /** 开头和 */ 结尾，可以包含多行注释和标签。

/**
 * 这是一个 JSDoc 注释的示例。
 */
